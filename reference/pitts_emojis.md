# Popular Emojis

The most popular Emoji of a given week in a given category from the
Meltwater Tweet sample. They can be rendered by using `"richtext"` with
[`annotate()`](https://ggplot2.tidyverse.org/reference/annotate.html).

## Usage

``` r
pitts_emojis
```

## Format

An object of class `character` of length 270.
